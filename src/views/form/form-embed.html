Embed Code

Here is the code to embed this form within your application. 
<formio src="'{{ apiProtocol }}://{{ currentProject.name }}.{{ apiServer }}/{{ form.path }}'"></formio>

How to Embed

Try one of these two ways to embed this form within your HTML application.

Quick Embed - iFrame

The iframe embed allows a self contained embed of this form into your application. Use this if you are embedding this within an existing CMS such as Wordpress or Drupal.

With this method, you will not have CSS access to the rendered form.

HTML Embed

For HTML embedding, you can either use Bower or add the script directly.

  • For Bower embedding 

    bower install --save ng-formio

    We recommend using a tool called Wiredep to automatically add all the dependencies to the <script> section.

  • For Script Tag embedding

    Add the following along with the other scripts on your HTML page.

    <script type="text/javascript" src="https://npmcdn.com/ng-formio@latest/dist/formio-complete.min.js"></script>

    You will need to also include both Angular.js (1.5) and jQuery as a dependency to your application to get this working.

Next, you will need to create an angular application and add formio as an included module. Add this below the include tag.

<script type="text/javascript">
  angular.module('formioApp', ['formio']);
  angular.bootstrap(document, ['formioApp']);
</script>

You can then add the <formio> tag anywhere within your HTML document, and it will render this form. Please go to https://help.form.io/api/#angular for more information.

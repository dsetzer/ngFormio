Install dependencies & Set Up

This step requires that that nodejs, npm and bower are installed. Bower can be installed after nodejs and npm by typing this in the terminal. Then install both node and bower dependencies by typing this in your terminal from the application base directory.

npm install -g bower

npm install

bower install

Configure for your project

In order for the app to use your form.io project, you need to configure the app to know where it is.

Change the first line of /src/config.js to say the following.

var APP_URL = '{{ projectUrl }}';

Running the app

You can run a development version of the application from the command line.

gulp serve

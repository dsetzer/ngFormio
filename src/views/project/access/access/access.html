Project Access Settings

These permissions control which users have access to form definitions themselves. To set who has access to form submissions, please use the Access tab on a form or resource.

Create All

The Create All permission allows users with the specified roles to create new forms in either the portal or your application.

Read All

The Read All permission allows users with the specified roles to read all forms. Please note that any user may view a specific form definition but this permission is required to access the form index at /form.

Update All

The Update All permission allows users with the specified roles to modify all existing forms in either the portal or your application.

Delete All

The Delete All permission allows users with the specified roles to delete all existing forms in either the portal or your application.

Warning

Please be very careful with these settings. Giving users access to modify or delete forms gives them almost administrative control of your app. Only assign them to trusted users.

To begin development of your application you will need to set up a local development environment. There are three ways to do this. You can start with our StarterKit, start with another application framework or roll your own.

With either option, you will need to make sure that you first install the Angular CLI tool by typing the following within your terminal prompt.

npm install -g @angular/cli

Now that you have the CLI tool enabled, you can do either of the following.

  • A
    Clone the Form.io Angular StarterKit

    Our Angular StarterKit comes set up with Angular Router, ngFormio and lots of Providers that make it easy to set up and use our forms within Angular. This is the easiest route to getting a running application.

    In order to clone the github repo you must have git installed on your computer. If you don't already have it, you can get it at https://git-scm.com/.

    Once you have git, go to your terminal and type

    git clone https://github.com/formio/angular-app-starterkit.git
cd angular-app-starterkit

    If you do no have git installed, you can go to https://github.com/formio/angular-app-starterkit and click the Download Zip button and unzip it on your computer.

    Finally, be sure to install all the dependencies to get everything set up. From within the application root directory type

    npm install
ng serve

  • B
    Use Angular CLI

    The Angular CLI tool allows you to easily create a new Angular application and rapidly add new modules and components.

    To set it up, go to your terminal and type

    ng new my-application --style=scss

    This will create a new application and then install all the depdencies. After it is done, you can then launch the application using.

    cd my-application
ng serve

    Next, add Bootstrap to your application.

    npm install --save bootstrap

    src/styles.scss 
    @import "~bootstrap/scss/bootstrap.scss";

    Now, add angular-formio.

    npm install --save angular-formio

    Then add the angular-formio module to your application by ensuring they are added to your module dependencies.

    src/app/app.module.ts 
    import { NgModule } from '@angular/core';
      import { FormioModule } from 'angular-formio';
      import { BrowserModule } from '@angular/platform-browser';
      import { MainComponent } from './main.component.ts';

      @NgModule({
         imports: [
            BrowserModule,
            FormioModule
         ],
         declarations: [
            MainComponent
         ],
         bootstrap: [
            MainComponent
         ]
      })
      export class AppModule {}

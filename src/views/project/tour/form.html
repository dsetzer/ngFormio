• To create a form, go to the
    Forms     page on the left and click on New Form

  • Select API Web Form to build a web form instead of a PDF Form.

  • Add a title to the form. Call it Demographics

  • Add the following components to the form:

    • A Text Field with the label First Name
    • A Text Field with the label Last Name
    • An Email Field with the label Email
    • A Text Area with the label Message

    Your form should look like this:

  • When done, click the Create Form button

  • Congratulations! You've created your first form on Form.io. This form definition is now stored as a JSON representation of the form and has a corresponding REST API for submissions.

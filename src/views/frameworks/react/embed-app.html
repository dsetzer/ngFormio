The simplest way to embed a Form.io form inside your app is to use the formio directive. There are lots of options for how to do this. Based on what you want to do, use one of the following:

  • A
    Uncontrolled Embed

    The easiest way to do an embed is to directly embed via the form URL.

    To embed a form within your react application, import the Formio library and add it to the render tree.

    import React from 'react';
      import ReactDOM from 'react-dom';
      import { Formio } from 'react-formio';

      ReactDOM.render(
        <Formio src="{{ variables.projectUrl }}/demographics" onSubmitDone={function(submission)} { // do something }" />, document.getElementById('example')
      );
  • B
    Controlled Embed

    A controlled embed is where the loading and saving are controlled outside of the actual element. Typically this is done with flux or redux.

    import React from 'react';
      import ReactDOM from 'react-dom';
      import { Formio } from 'react-formio';

      // These are typically loaded from flux or redux in a controlled component.
      var form = {};
      var submission = {
        data: {}
      };

      ReactDOM.render(
        <Formio form={form} submission={submission} onChange={function() {}} onSubmit={function() {}} />, document.getElementById('example')
      );

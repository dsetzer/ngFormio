Welcome to

Form.io is a platform that enables developers to rapidly build modern web and mobile applications. Our intuitive drag & drop interface allows you to create both the forms and the RESTful API's all in one easy step! You build the app, we handle the Form I/O.

Getting started is easy.

Use our command line utility to bootstrap a working application.

npm install -g formio-cli

formio bootstrap formio/formio-app-todo

Note: You must have Node.js installed on your machine.

Other applications can be found @ https://github.com/formio

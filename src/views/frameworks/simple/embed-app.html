When embedding the form into a webpage, you have two options for how to do it. You can directly use our formio.js javascript library or use iframes. The javascript library has more dependencies and more features and the iframe is simpler but more limited in its functionality.

  • A
    Embed using the formio.js javascript library

    You can use the formio.js library as a JavaScript embedding of a form using a single line of code. For example, you can simply use the following embed code.

    <script src="https://unpkg.com/formiojs@latest/dist/formio.embed.min.js?src={{variables.projectUrl}}/demographics" />

    For an example of how this looks and works, check out the following Form.io Form Embedding CodePen
  • B
    Embed using iframes

    The iframe embed allows a self contained embed of this form into your application. Use this if you are embedding this within an existing CMS such as Wordpress or Drupal.

    With this method, you will not have CSS access to the rendered form.

Create an Admin User account

Your project came pre-configured with three roles: Administrator, Authenticated, and Anonymous. In order to begin using your application, you will need to create an admin user within your project so that you can log in and have the correct permissions. You can adjust the Access for your project by going to the Access settings.

Submit the following form to create an Admin. You can also go to Resources -> Admin -> Use to create additional Admin users.

You can use this account to log in to your application for the first time.

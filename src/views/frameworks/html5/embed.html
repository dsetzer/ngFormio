Include Formio.js + Bootstrap

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">
    <link rel="stylesheet" href="https://unpkg.com/formiojs@latest/dist/formio.full.min.css">
    <script src="https://unpkg.com/formiojs@latest/dist/formio.full.min.js"></script>

Embed the HTML Element

Place the following where you would like to show this form.

<div id="formio"></div>

Attach the form to the element

Next, you will attach the form to the HTML element.

<script type="text/javascript">
        window.onload = function() {
          Formio.createForm(document.getElementById('formio'), '{{ projectUrl }}/{{ form.path }}');
        };
      </script>

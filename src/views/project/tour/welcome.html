Welcome to Your New Project!

This walkthrough will guide you through starting to build an application with Form.io

First, get oriented with the Form.io portal.

Project Settings

Project settings can be found in the light grey bar at the top of this page.

  • Trial
    You can upgrade your project to a different plan and set your billing information.
  • Teams
    Teams allow sharing this project with other developers on Form.io.
  • The project settings such as the project name and path.
Project Content

In the dark blue strip on the left hand side of the page you will find the contents of your project.


  • Resources
    Resources are objects that can be used to create complex data structures within your application. Resources can be referenced within other forms to create complex nested resource relationships. Each resource has a corresponding form for embedding.

  • Forms
    Forms are used to build the foundations of your application. They are built using a drag and drop form builder. When you build a form, an API is automatically created for it as well.

  • API
    All forms and resources also have an automatically generated REST API that can be found in the API page. These are automatically attached to embedded forms and can also be used independently with a REST library.

  • Data
    Data that is sent to the API can be browsed using the Data page. You can browse, edit, create and download all the data submitted to the forms.

  • Access
    Each Form.io project comes with a full Role Based Access Control system. You can configure who has access to what forms and the data submitted to those forms using the Access page.

  • Settings
    The settings section is where you can change integration keys and control staging and remote environments for your project's stage.

Install react-formio

Install the react-formio module.

npm install --save react-formio

Render the directive

To embed a form within your react application, import the Formio library and add it to the render tree.

import React from 'react';
      import ReactDOM from 'react-dom';
      import { Form } from 'react-formio';

      ReactDOM.render(
        <Form src="{{ projectUrl }}/{{ form.path }}" />, document.getElementById('example')
      );

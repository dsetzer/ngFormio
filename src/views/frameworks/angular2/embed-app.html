The simplest way to embed a Form.io form inside your app is to use the formio directive. There are lots of options for how to do this. Based on what you want to do, use one of the following:

  • A
    Uncontrolled Embed

    Include the <formio> component within a component

    main.component.ts 
    import { Component } from '@angular/core';

@Component({
  template: '<formio src="{{ variables.projectUrl }}/demographics" />'
})

export class MainComponent {}
  • B
    Controlled Embed

    This is similar to the uncontrolled embed but we are going to load the form first and then pass it to the formio directive. 

    import { Component } from '@angular/core';

        @Component({
          template: '<formio form="form" />'
        })

        export class MainComponent {}

    In this case the form scope variable is loaded by a controller using the formio.js library.

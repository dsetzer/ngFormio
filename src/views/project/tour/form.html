• 1
    Go to form page

    To create a form, go to the
    Forms     page on the left and click on New Form

    Then select API Web Form to build a web form instead of a PDF Form.

  • 2
    Build the form

    Add a title to the form. Call it Demographics.

    Add the following components to the form:

    • A Text Field with the label First Name
    • A Text Field with the label Last Name
    • An Email Field with the label Email
    • A Text Area with the label Message

  • 3
    Save and test the form

    When done, click the Create Form button.

    You should see a message saying you have successfully saved the form.

    Go to the Use tab to test it out.

Congratulations! You've created your first form on Form.io. This form definition is now stored as a JSON representation of the form and has a corresponding REST API for submissions. In a couple steps we'll look at how to embed it within your application. But first, lets add an email action to it.

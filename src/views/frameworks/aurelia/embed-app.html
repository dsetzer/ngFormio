The simplest way to embed a Form.io form inside your app is to use the formio component.

  • A
    Use the formio component in a template.

    Add the following in one of your templates.

    <template>
  <formio src="https://examples.form.io/example" submission.bind="submission"></formio>
</template>
  • B
    Add properties to your class

    You can bind properties of the component by setting them on your class.

    export class MyClass {
  private submission: any = {
    data: {
        firstName: 'Joe',
        lastName: 'Smith'
    }
  };
}

About Projects

A project serves as a container for the Resources, Forms, Data, and Settings for a specific application you are building. A project can serve as the backend API platform for your application.

All of the Resources and Forms serve as the API platform as well as Form interface for your web application. Here is what you need to know to get started.

Resources

Form based data structures that can serve as the backend storage and API platform for your web application. Resources can also be embedded into other Resources and Forms to create a hierarchical relationship structure.

Forms

Serve as the form interfaces for your application. You can re-use Resource fields as well as use custom input components. The forms you create can range from simple surveys to complex authentication and resource management forms.

Data

Where you can find and manage all the data within your appliction. Here you can view charts of analytics data on how many submissions your users are making over time. You can also access an interactive table of all submissions for your forms that you can sort, filter, and more.

Preview

Preview your working application live. Any changes made in your Forms and Resources will immediately be reflected in the preview application. You may also as download the preview application as a base for Local Development

API

Provides an interactive API documentation explorer powered by Swagger.IO.

• 1
    Create the mount point

    Place the following where you would like to show the form.

    <div id="formio" />
  • 2
    Attach the form to the mount point

    Next, you will attach the form to the HTML element.

    <script type="text/javascript">
        window.onload = function() {
          Formio.createForm(document.getElementById('formio'), '{{ variables.projectUrl }}/demographics');
        };
      </script>

The form should now render on your page. If you try it out and receive an access denied error, be sure to go to the access settings for the form and allow anonymous users to create submissions.

The simplest way to embed a Form.io form inside your app is to use the formio directive. There are lots of options for how to do this. Based on what you want to do, use one of the following:

  • A
    Hosted Form Embed

    Include the <formio> component within a component

    main.component.ts 
    import { Component } from '@angular/core';

@Component({
  template: '<formio src="{{ variables.projectUrl }}/demographics" (submit)="onSubmit($event.value)" />'
})
export class MainComponent {
  onSubmit(submission) {
    alert(submission);
  }
}
  • B
    JSON Form Embed

    This is similar to the hosted form embed but we are going to pass the JSON form object directly to the formio directive. 

    import { Component } from '@angular/core';

@Component({
  template: '<formio form="{
    components: [
      {
        type: 'textfield',
        label: 'First Name',
        key: 'firstName',
        input: true
      },
      {
        type: 'textfield',
        label: 'Last Name',
        key: 'lastName',
        input: true
      }
    ]
  }" (submit)="onSubmit($event.value)" />'
})
export class MainComponent {
  onSubmit(submission) {
    alert(submission);
  }
}

Embed Code

Here is the code you will use to embed this form within your application. 
<formio src="'https://{{ currentProject.name }}.form.io/{{ form.path }}'"></formio>

How to Embed

Try one of these two ways to embed this form within your HTML application.

HTML Embed

It is recommended to use this method if you do not already have Angular.js installed within your web application.

First, you need to add the Form rendering library to your application within a <script> tag

<script type="text/javascript" src="https://cdn.rawgit.com/formio/ngFormio/develop/dist/formio-full.min.js"></script>

Next, you will need to create an angular application and add formio as an included module. Add this below the include tag.

<body ng-app="formApp">
        ...
        <script type="text/javascript">
        angular.module('formApp', ['formio']);
        </script>

You can then add the <formio> tag anywhere within <body> tag and it will render this form!

Bower

To add this form to your Application, use Bower to download the Form.IO library and its dependencies.

bower install --save formio

Next, ensure your webpage has all of the Project dependencies added to the <script> tag. For a simple example, refer to the following example.

Example Simple Form Embedding

We recommend using a tool called Wiredep to automatically add all the dependencies to the <script> section.

Then you need to add formio as a dependency to your application's Angular module

<body ng-app="formApp">
        ...
        <script type="text/javascript">
        angular.module('formApp', ['formio']);
        </script>

You can then add the <formio> tag anywhere within <body> tag and it will render this form!

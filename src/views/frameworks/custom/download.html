To begin development of your application you will need to set up a local development environment. Since you are doing this your own custom way, the details here are very generic.

  • 1
    Add the formio.js library to your application

    The formio.js library is a Javascript only library that renders form.io forms. You can use it to render within any webpage.

    Either use a package manager like npm or bower to include the formio.js library or include it manually.

    You can also check out one of our many framework libraries.

    Formio.js library

  • B
    Include formio.js in your webpage

    Be sure to include both the javascript and the CSS or else the form will look odd. These can either be from a CDN or your local package manager.

    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">
<link rel="stylesheet" href="https://unpkg.com/formiojs@latest/dist/formio.full.min.css">
<script src="https://unpkg.com/formiojs@latest/dist/formio.full.min.js"></script>

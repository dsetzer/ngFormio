Local Development

This application can be ran locally by downloading the GitHub repo located here.

{{ repo }}

Then change the /src/config.js to say the following.

var APP_URL = '{{ projectApi }}';

Then you can run the app by typing the following in your terminal within the directory of the download.

npm install;
bower install;
gulp serve;

Then add any form using the following syntax.

<formio src="'https://{{ currentProject.name }}.form.io/yourform'"></formio>

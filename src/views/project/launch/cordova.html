Local Development

In order to deploy your app in cordova, you first need to set it up locally. See Launch Local App Development.

Build a Distribution

To build your app and prepare it for cordova, you will need to compile it. If you are using one of our applications as the base, we provide a gulp task that will do this for you.

gulp build

This will build your entire application into the /dist directory. This will be a fully working version of your app that is ready for deployment.

Adding to Cordova

Adding the built files to your cordova project is as easy as copying the /dist files into the www directory within your cordova project.

Setting up your cordova project and compiling it is covered in more details at Apache Cordova with Form.io.

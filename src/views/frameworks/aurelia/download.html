To begin development of your application you will need to set up a local development environment. There are three ways to do this. You can start with our StarterKit, start generate an application or roll your own.

Your options are:

  • A
    Clone the Form.io Aurelia StarterKit

    Our React StarterKit comes set up with React Router, redux and react-formio that make it easy to set up and use our forms within React. This is the easiest route to getting a running application.

    In order to clone the github repo you must have git installed on your computer. If you don't already have it, you can get it at https://git-scm.com/.

    Once you have git, go to your terminal and type

    git clone https://github.com/formio/aurelia-app-starterkit.git
cd aurelia-app-starterkit

    If you do no have git installed, you can go to https://github.com/formio/aurelia-app-starterkit and click the Download Zip button and unzip it on your computer.

    Finally, be sure to install all the dependencies to get everything set up. From within the application root directory type

    npm install

  • B
    Use a Aurelia Generator

    You can get more information at https://aurelia.io/docs/tutorials/creating-a-contact-manager#setting-up-your-machine

    To set it up, go to your terminal and type

    npm install aurelia-cli -g

    Once the dependencies installed, simply launch the generator and answer the questions.

    au new

    Answer the questions as you want and be sure to install dependencies as part of the setup or run npm install on your own.

    Finally, add in the Aurelia Form.io library.

    • aurelia-formio - The main Aurelia form renderer.

    To add it to your application, type the following

    npm install --save aurelia-formio

    Be sure to add aurelia-formio as a plugin as well.

    export function configure(aurelia: Aurelia) {
  aurelia.use
    .standardConfiguration()
    .plugin(PLATFORM.moduleName('aurelia-formio'));

  • C
    Roll your own application

    If you are have experience with and are comfortable setting up your own Aurelia application you can, of course, do so. This works best for teams that already have an application or have a different set of build tools or dependent libraries than the starterkits use.

    To add Form.io forms to your application, use one of the package managers and include the aurelia-formio library to your application.

    • aurelia-formio - The main Aurelia form renderer.

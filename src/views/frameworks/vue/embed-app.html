The simplest way to embed a Form.io form inside your app is to use the formio directive. There are lots of options for how to do this. Based on what you want to do, use one of the following:

  • A
    Uncontrolled Embed

    The easiest way to do an embed is to directly embed via the form URL.

    To embed a form within your vue application, import the Formio library and add it to the render tree.

    
      <template>
        <formio src="{{ variables.projectUrl }}/demographics" v-on:submit="onSubmitMethod" />
      </template>
      <script>
      import { Formio } from 'vue-formio';

      export default {
        components: {
          formio: Formio
        },
        methods: {
          onSubmitMethod: function(submission) {
            console.log(submission);
          }
        }
      };
      </script>
  • B
    Controlled Embed

    A controlled embed is where the loading and saving are controlled outside of the actual element. Typically this is done with flux or redux.

    
        <template>
          <formio :form="form" :submission="submission" v-on:change="onChangeMethod" v-on:submit="onSubmitMethod" />
        </template>
        <script>
        import { Formio } from 'vue-formio';

        export default {
          data: function() {
            // Load these from vuex or some other state store system.
            return {
              form: myForm,
              submission: mySubmission
            }
          },
          components: {
            formio: Formio
          },
          methods: {
            onSubmitMethod: function(submission) {
              console.log(submission);
            },
            onChangeMethod: function(change) {
              console.log(change);
            }
          }
        };
        </script>

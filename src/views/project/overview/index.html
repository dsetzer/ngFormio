Application

How easy can you create an Application with us?

As easy as following these steps

Application: Download

To begin development of this application you will need to set up a local development environment. Start by either downloading or cloning the github repo.

Download the repo

Downloading the repo does not require git but also does not track and manage changes you make unless you set it up manually with a new git repo.

Go to {{ repository }}

Click the Download Zip button.

Unzip the file, open your terminal and change directory to the app directory.

Clone the repo

In order to clone the github repo you must have git installed on your computer. If you don't already have it, you can get it at https://git-scm.com/.

Once you have git, go to your terminal and type

git clone {{ repository }}.git
cd {{ appFolder }}

Application: Set Up

Install dependencies & Set Up

This step requires that that nodejs, npm and bower are installed. Bower can be installed after nodejs and npm by typing this in the terminal. Then install both node and bower dependencies by typing this in your terminal from the application base directory.

npm install -g bower

npm install

bower install

Configure for your project

In order for the app to use your form.io project, you need to configure the app to know where it is.

Change the first line of /src/config.js to say the following.

var APP_URL = '{{ projectApi }}';

Running the app

You can run a development version of the application from the command line.

gulp serve

Application Setup: Account

Create an account

Now you have on your local an application with Form.io

Form.io provides you with an authentication process that is configured when creating a Resource or a Form, these are called Form Actions.

Now that we have our running application, let's build a Form with Actions

Form: Create

To create a form, we have to go to the Forms page and click on Create Form Choose API Web Form, Let's name the form Email Action and set an API path

With the Drag-n-Drop components select the Special Components and Drag the Email Component, then let's put a Label and a Placeholder to our field component. Finally, hit save and then click Create Form.

Congrats! we have created an API Web Form, now let's add Actions to this form .

Form: Add Action

To add an email action you have to go to the actions page in your form and select the email action, then click on Add Action

This will create an action when a form is submitted, let's send an email notification from no-reply@form.io to the email that we submit in our form. We are going to use our data object to select the field component that is called "email" and then we can write any message to send.

That's it! we set an Email Action to our API Web Form, in this case every time that our form is submitted we send an Email. Now, how we put this form in our app?.

React

Include the react-formio library

Be sure you have added the react-formio libary to your application.

Render the directive

To embed a form within your react application, import the Formio library and add it to the render tree.


          import React from 'react';
          import ReactDOM from 'react-dom';
          import { Formio } from 'react-formio';

          ReactDOM.render(
            <Formio src="{{ projectPath(currentProject) }}/email" />,document.getElementById('example')
          );

Angular 1.x

Include the ngFormio module

Be sure you have added the ngFormio library to your application and have included it in the modules list in the app declaration.

Embed within a template

To embed a form within your angular 1.x application, add the formio directive within a template.


          angular.module('yourApp', [
            'formio'
          ]);

          ---

          <formio src="'{{ projectPath(currentProject) }}/email'" />

Vue.JS

Creating a Vue component

Be sure you have added the formiojs library to your application and imported it in the app declaration.

Embed within a template

To embed a form within your Vue application, lets create a component call FormVue, first you have to import FormioForm from the formiojs library.


          // FormVue.vue
          <template>
              <div id="formio"> </div>
          </template>
          <script>
            import FormioForm from 'formiojs/form';
              export default {
                name: 'formio-vue',
                props: ['src'],
         mounted () {
                let self = this;
             let form = new FormioForm(document.getElementById('formio'));
            form.src = self.src;
            form.on('submit', (submission) => {
               console.log('The form was just submitted!!!');
          });
           form.on('error', (errors) => {
                console.log('We have errors!');
          })
          }
      }
  </script>

          ---
          // Then we can use our component in our App.vue.
                <template>
             <div id="app">
             <formio-vue src="'{{ projectPath(currentProject) }}/email'"></formio-vue>
           </div>
          </template>
          <script>
		  import FormioVue from './components/Formio.vue'
      export default {
          name: 'app',
          components: {
              FormioVue
        }
      }
   </script>

Angular 2.x

Include the ng2-formio module

Be sure you have added the ng2-formio library to your application and have included it in the modules list in the app declaration.

Embed within a template

To embed a form within your angular 2.x application, add the formio directive within a template.


          import { FormioModule } from 'ng2-formio';
          @NgModule({
            imports: [ BrowserModule, ReactiveFormsModule, FormioModule ],
            declarations: [ AppComponent ],
            bootstrap: [ AppComponent ]
          })
          export class AppModule { }

          ---

          <formio src='{{ projectPath(currentProject) }}/email' />

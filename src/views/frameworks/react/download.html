To begin development of your application you will need to set up a local development environment. There are three ways to do this. You can start with our StarterKit, start with another application framework or roll your own.

Your options are:

  • A
    Clone the Form.io React StarterKit

    Our React StarterKit comes set up with React Router, ngFormio and lots of Providers that make it easy to set up and use our forms within React. This is the easiest route to getting a running application.

    In order to clone the github repo you must have git installed on your computer. If you don't already have it, you can get it at https://git-scm.com/.

    Once you have git, go to your terminal and type

    git clone git@github.com:formio/react-app-starterkit.git
cd react-app-starterkit

    If you do no have git installed, you can go to https://github.com/formio/react-app-starterkit and click the Download Zip button and unzip it on your computer.

    Finally, be sure to install all the dependencies to get everything set up. From within the application root directory type

    npm install

  • B
    Use a Fountain.js Generator

    Fountain.js is a yeoman generator that will build the framework of an application for you and has a lot of sane defaults. It is a good way to get started but does not come pre-configured with the Form.io libraries installed.

    To set it up, go to your terminal and type

    npm install -g yo generator-fountain-webapp

    Once the dependencies installed, simply launch the generator and answer the questions.

    mkdir myapp
cd myapp
yo fountain-webapp

    You can choose whatever you want for the options but we recommend the following

    ? Which JavaScript framework do you want? React
? Which module management do you want? Webpack with NPM
? Which JS preprocessor do you want? ES2015 today with Babel
? Which CSS preprocessor do you want? SASS
? Which Continuous Integration platform do you want?
? Do you want a sample app? A working landing page
? Would you like a router? React router

    Be sure to install all the dependencies to get everything set up. From within the application root directory, type these commands

    npm install

    Finally, add in the React Form.io library.

    • react-formio - The main React form renderer.

    To add it to your application, type the following

    npm install --save react-formio

  • C
    Roll your own application

    If you are have experience with and are comfortable setting up your own React application you can, of course, do so. This works best for teams that already have an application or have a different set of build tools or dependent libraries than the starterkits use.

    To add Form.io forms to your application, use one of the package managers and include the react-formio library to your application.

    • react-formio - The main React form renderer.

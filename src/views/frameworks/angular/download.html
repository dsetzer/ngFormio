To begin development of your application you will need to set up a local development environment. There are three ways to do this. You can start with our StarterKit, start with another application framework or roll your own.

Your options are:

  • A
    Clone the Form.io AngularJS StarterKit

    Our AngularJS StarterKit comes set up with Angular UI Router, ngFormio and lots of Providers that make it easy to set up and use our forms within AngularJS. This is the easiest route to getting a running application.

    In order to clone the github repo you must have git installed on your computer. If you don't already have it, you can get it at https://git-scm.com/.

    Once you have git, go to your terminal and type

    git clone https://github.com/formio/ng-app-starterkit.git
cd ng-app-starterkit

    If you do no have git installed, you can go to https://github.com/formio/ng-app-starterkit and click the Download Zip button and unzip it on your computer.

    Finally, be sure to install all the dependencies to get everything set up. From within the application root directory, type these commands

    npm install
bower install

  • B
    Use a Fountain.js Generator

    Fountain.js is a yeoman generator that will build the framework of an application for you and has a lot of sane defaults. It is a good way to get started but does not come pre-configured with the Form.io libraries installed.

    To set it up, go to your terminal and type

    npm install -g yo generator-fountain-webapp

    Once the dependencies installed, simply launch the generator and answer the questions.

    mkdir myapp
cd myapp
yo fountain-webapp

    You can choose whatever you want for the options but we recommend the following

    ? Which JavaScript framework do you want? Angular 1
? Which module management do you want? None with Bower and script injection
? Which JS preprocessor do you want? Pure old JavaScript
? Which CSS preprocessor do you want? SASS
? Which Continuous Integration platform do you want?
? Do you want a sample app? A working landing page
? Would you like a router? Angular UI Router

    Be sure to install all the dependencies to get everything set up. From within the application root directory, type these commands

    npm install
bower install

    Finally, add in the Form.io libraries that you want.

    • ngFormio - The main AngularJS form renderer.
    • ngFormBuilder - The AngularJS form builder.
    • ngFormioHelper - The AngularJS helper library that includes providers for Authentication, Form Builder and more.

    To add these to your application, type the following

    bower install --save ng-formio ng-formio-builder ng-formio-helper

    Then add them to your application by ensuring they are added to your module dependencies.

    app.module('myapp', ['formio', 'ngFormBuilder', 'ngFormioHelper']);

  • C
    Roll your own application

    If you are have experience with and are comfortable setting up your own AngularJS application you can, of course, do so. This works best for teams that already have an application or have a different set of build tools or dependent libraries than the starterkits use.

    To add Form.io forms to your application, use one of the package managers and include the ngFormio library to your application. If you would like to do form building inside your app as well, include the ngFormBuilder library as well.

    • ngFormio - The main AngularJS form renderer.
    • ngFormBuilder - The AngularJS form builder.

    Then add them to your application by ensuring they are added to your module dependencies.

    app.module('myapp', ['formio', 'ngFormBuilder']);
